1、tinxie.py从tinxie中读取词语列表，然后百度词语网站中获取数据。然后在out目录下生成output.json。output.json包含anki字典的词条数据。如果有无法识别的词条会记录在error.json中。

2、语文_听写.json是生成好的anki词典文件，通过CrowdAnki插件导入或导出。
